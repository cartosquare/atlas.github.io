深度学习框架 caffe 简介

发表于 2016-10-19   |   分类于 深度学习   |     |   阅读次数

caffe 是一个视觉领域方面的深度学习框架，以配置简单，运行高效著称。

caffe术语

Blobs

caffe 用来存储训练数据（图片）、模型参数和优化导数等数据的的数据结构，实际上是一个多维数组。
比如，用来存储图片的 blob 的维度为（N,K,H,W)。N表示图片的数目，K表示图片的波段数，H和W分别表示图片的高和宽。

Layers

Layers 是基本的计算单元。即 caffe 的所有的基本操作都是针对Layer的。比如，可以对一个 Layer 进行卷积操作、池化操作，应用诸如 sigmoid 函数等的线性变换，进行正规化操作等等。

每个 Layer 都要定义三个过程：

  • 初始化：训练开始时初始化层的所有参数和连接
  • 前向迭代：给定 bottom Layer，如何计算出 output Layer
  • 后向迭代：给定 output Layer 的导数，如何计算导数并传递给 bottom Layer

Net

Net 通过组合连接 Layers 形成了一个函数。这个函数为了完成某项具体的任务（比如分类、分割等）；而 Net 通过自动微分Layers 得到的导数则定义了学习这个任务的方法。整个Net就是一个端到端的学习引擎。
一个典型的Net是许多Layers的组合。首先从一个Data Layer加载训练数据，最后用一个loss Layer来计算针对某项任务的损失函数（目标函数）

Solver

通过前向和后向迭代，更新模型参数，以最小化损失函数。

  • Stochastic Gradient Descent (type: “SGD”),
  • AdaDelta (type: “AdaDelta”),
  • Adaptive Gradient (type: “AdaGrad”),
  • Adam (type: “Adam”),
  • Nesterov’s Accelerated Gradient (type: “Nesterov”) and
  • RMSprop (type: “RMSProp”)
